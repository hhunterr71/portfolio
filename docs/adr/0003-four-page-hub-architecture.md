# Four-page architecture with a minimal-nav Hub

The site previously read as three loosely related sections stitched into one anchor-scrolling page. We're splitting it into four pages: a minimal-nav Hub at `/` (intro + links to the other three, with no section treated as "more home" than another) plus Resume, Projects, and Creative, each carrying a persistent shared top nav. A single rebuilt scrolling page was considered, but it would have kept the same "resume is the real site, everything else is an appendix" imbalance the redesign is meant to fix.
