# Use Tailwind CSS instead of Bootstrap

The current site's Bootstrap defaults (navbar, accordion, carousel) make it read as a generic template rather than a cohesive personal site. We're using Tailwind CSS with a small custom design-token palette (color/type/spacing) instead, so all three content areas draw from one deliberate visual language rather than Bootstrap's component opinions layered with custom CSS on top. Plain hand-written CSS was considered and rejected only because Tailwind gives the same custom-look outcome with faster day-to-day iteration.
